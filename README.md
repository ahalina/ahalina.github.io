# 🍔 Vintage Food

Современный лендинг для ресторана быстрого питания с красивым дизайном, адаптивной вёрсткой и корзиной заказов.

## 📋 Описание проекта

Vintage Food — это веб-сайт ресторана, который сочетает в себе традиционные рецепты и современный подход к подаче. Сайт содержит информацию о меню, акциях, доставке и даёт возможность ознакомиться с ассортиментом.

## ✨ Функциональность

- 🏠 **Главный экран** с призывом сделать заказ и получить скидку
- 📋 **Меню** с категориями блюд (бургеры, курица, горячие блюда, закуски, салаты, десерты)
- 🛒 **Модальное окно корзины** (открывается по клику на иконку корзины)
- 🎁 **Боковая панель** с акциями и спецпредложениями
- 📱 **Адаптивный дизайн** для мобильных устройств
- 🍔 **Выпадающее подменю** для раздела "Menu"
- 🔗 **Социальные кнопки** (VK, Telegram)
- 📞 **Футер** с юридической информацией и ссылками

## 🛠️ Технологии

- **HTML5** — семантическая структура страницы
- **CSS3** — стилизация, градиенты, анимации, адаптивность (Flexbox, Grid, медиазапросы)
- **JavaScript (ES6)** — открытие/закрытие модального окна корзины
